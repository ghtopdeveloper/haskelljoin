##Iso
[Data.Text.Lens](https://hackage.haskell.org/package/lens/docs/Data-Text-Lens.html)

```haskell
packed :: IsText t => Iso' String t

pack x ≡ x ^. packed
unpack x ≡ x ^. from packed
```
