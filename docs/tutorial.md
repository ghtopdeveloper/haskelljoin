<summary>エラトステネスの篩</summary>

```haskell
primes = sieve [2..]
sieve (p:xs) = p : sieve [x | x <- xs, x `mod` p /= 0]

-- ghci> primes
-- [2,3,5,7,11,13,17,19,23,..
```

<summary>FizzBuzz</summary>

```hs
let (m ~> str) x = str <$ guard (x `mod` m == 0)
in map (fromMaybe . show <*> 3 ~> "fizz" <> 5 ~> "buzz")
```
